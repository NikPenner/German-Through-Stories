Aufgabe 7. Persönliche Fragen.

Answer each question with a full sentence. Make sure to use ‘gern’. Try to start some sentences with the underlined element: “Kaffee trinke ich sehr gern.”

1. Hören Sie gern Musik von Taylor Swift?
2. Was machen Sie gern am Wochenende?
3. Was machen Sie nicht gern am Freitagabend?
4. Was essen Sie nicht gern?
